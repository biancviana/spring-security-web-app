# 🔐 Secure Web App - Spring Boot

Aplicação web desenvolvida com foco em **segurança**, utilizando **Spring Boot**, **Spring Security** e **MySQL**.

Este projeto foi criado como parte da disciplina de **Projeto de Segurança da Informação**, implementando práticas baseadas no **OWASP Top 10**.

---

## 🚀 Funcionalidades

- 🔑 Autenticação de usuários (login)
- 👑 Controle de acesso por papéis (ADMIN / USER)
- ➕ Cadastro de usuários (restrito a administradores)
- 🔒 Áreas protegidas por tipo de usuário
- 🚪 Logout seguro com invalidação de sessão

---

## 🛡️ Mecanismos de Segurança

- ✔ Proteção contra **CSRF (Cross-Site Request Forgery)**
- ✔ Prevenção de **SQL Injection** com JPA
- ✔ Controle de acesso baseado em papéis (RBAC)
- ✔ Hash de senhas com **BCrypt**
- ✔ Gerenciamento seguro de sessão
- ✔ Tratamento de erros sem exposição de detalhes internos

---

## 🧑‍💻 Tecnologias Utilizadas

- Java 17+
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Thymeleaf (templates HTML)
